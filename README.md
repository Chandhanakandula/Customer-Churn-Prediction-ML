# Customer Churn Prediction using Machine Learning

## Description
This project predicts whether a customer will churn using supervised machine learning classification techniques.

## Techniques Used
- Supervised Machine Learning
- Classification Techniques
- Logistic Regression
- Random Forest

## Dataset
- Telco Customer Churn Dataset (Kaggle)

## Tools & Technologies
- Python
- Pandas
- Scikit-learn

## Output
- Logistic Regression Accuracy: ~80%
- Random Forest Accuracy: ~85%

## Conclusion
The model helps identify customers who are likely to leave the service, enabling better business decisions.
